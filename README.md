# 💰 Accountant Bot

Telegram-бот для учёта расходов в групповом чате. Парсит сообщения, распределяет по категориям и формирует отчёты.

---

## 🚀 Быстрый старт

### 1. Установите зависимости

```bash
pip install python-telegram-bot==20.7 apscheduler pytz
```

### 2. Создайте бота в Telegram

1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Команда `/newbot` → придумайте имя и username
3. Скопируйте токен (выглядит как `1234567890:ABCdef...`)

### 3. Настройте переменные окружения

```bash
export BOT_TOKEN="ВАШ_ТОКЕН_ЗДЕСЬ"
export TARGET_CHAT_ID="0"   # Заполните после шага 5
```

Или просто вставьте токен прямо в `bot.py`:

```python
BOT_TOKEN = "ВАШ_ТОКЕН_ЗДЕСЬ"
```

### 4. Запустите бота

```bash
python bot.py
```

### 5. Добавьте бота в группу

1. Откройте группу в Telegram → Добавить участника → найдите вашего бота
2. **Важно:** дайте боту права читать сообщения (по умолчанию есть)
3. Напишите в группе `/chatid` — бот ответит ID чата
4. Вставьте ID в `TARGET_CHAT_ID` в коде или переменной окружения
5. Перезапустите бота

## 📝 Форматы записи расходов

Бот понимает любой порядок числа и описания:

```
1000 такси
20 тыс куртка
42 700 - продукты
потратил 5000 на еду
кофе 350
1.5к бензин
куртка 20000
```

---

## 📊 Команды

| Команда    | Действие                 |
| ---------- | ------------------------ |
| `/week`    | Отчёт за текущую неделю  |
| `/month`   | Отчёт за текущий месяц   |
| `/quarter` | Отчёт за текущий квартал |
| `/chatid`  | Узнать ID текущего чата  |
| `/start`   | Приветствие и справка    |

## 🗂️ Категории

Бот автоматически определяет категорию по ключевым словам:

- 🚕 **Транспорт** — такси, убер, бензин, парковка...
- 🛒 **Продукты** — еда, магазин, рынок, супермаркет...
- 🍽️ **Кафе и рестораны** — кафе, обед, кофе, доставка...
- 👗 **Одежда** — куртка, обувь, джинсы...
- 💊 **Здоровье** — аптека, врач, лекарства...
- 🏠 **Дом и ЖКУ** — аренда, коммуналка, интернет...
- 🎮 **Развлечения** — кино, подписки, фитнес...
- 📚 **Образование** — курсы, книги...
- ✂️ **Красота** — парикмахер, маникюр...
- 💸 **Прочее** — всё остальное

---

## ⚠️ Важно: данные в памяти

Бот хранит расходы **в памяти** пока работает. При перезапуске данные сбрасываются.

**Если нужно сохранять данные между перезапусками** — в будущем можно добавить SQLite (1 файл, без сервера). Напишите `/month` в чате — бот сформирует отчёт по накопленным за сессию данным.
